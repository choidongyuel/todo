# 페르소나 문서 (02-personas.md)

## 주 페르소나: 효율적인 직장인 "민준"

### 프로필
- **나이/직업:** 28세, 회사원 (마케팅 팀)
- **특징:** 동시에 여러 프로젝트 진행, 시간 관리 능력이 뛰어남
- **디바이스 주 사용:** 모바일 (70%), 데스크톱 (30%)

### 목표
- 많은 업무 중에서 **중요한 것을 빠뜨리지 않기**
- 업무 우선순위를 명확히 하여 **집중도 높이기**
- 재작업/추적 없이 **효율적으로 일 진행하기**

### 현재 불만
- **Notion:** 너무 무겁고, 매번 새 항목 추가하는데 5초 이상 소요
- **Google Tasks:** 기능이 부족해서 카테고리 관리가 안 됨
- **Todoist:** 기능이 너무 많아서 오버킬, 프리 플랜은 제약이 많음
- **메모장:** 체계적이지 않고, 완료 상태 추적이 어려움

### 행동 패턴
| 시간대 | 행동 | 목적 |
|------|------|------|
| 아침 9시 | 메일 확인 후 오늘의 업무를 빠르게 리스트업 | 우선순위 파악 |
| 회의 전후 | 모바일로 빠르게 확인하고 완료 마크 | 상태 동기화 |
| 점심시간 | 모바일로 오후 업무 확인, 새 요청 사항 추가 | 일정 조정 |
| 퇴근 전 | 오늘 못한 일을 내일로 미루기, 완료 항목 정리 | 심리적 완결 |

### 기대하는 앱의 모습
- **빠름:** 모바일에서 3초 이내에 새 할 일 추가 가능
- **깔끔함:** 복잡한 설정 없이 기본값으로 바로 쓸 수 있음
- **시각적:** 우선순위와 카테고리로 한눈에 중요도 파악
- **항상 손에:** 모바일에서 수시로 빠르게 접근 가능

## 대조 페르소나: 체계적인 대학생 "지은"

### 프로필
- **나이/신분:** 21세, 대학교 3학년 (인문계열)
- **특징:** 계획을 세워서 실행하는 스타일, 장기 프로젝트 추적 능력 뛰어남
- **디바이스 주 사용:** 데스크톱 (50%), 모바일 (50%)

### 목표
- **중요한 과제/시험 기한을 놓치지 않기**
- **일주일 단위로 공부 시간 계획하고 추적하기**
- **동아리 활동과 학업을 한 곳에서 관리하기**

### 현재 불만
- **여러 앱 필요:** 캘린더, 메모, 체크리스트를 따로 써야 해서 복잡
- **모바일 앱 번거로움:** 스마트폰에 앱을 너무 많이 설치하면 배터리 소진
- **정보 산산이:** 과제는 학교 LMS, 동아리는 카톡, 개인 일정은 캘린더... 통합이 안 됨

### 행동 패턴
| 시간대 | 행동 | 목적 |
|------|------|------|
| 주말 저녁 | 노트북으로 한 주 계획 세우기 (과제, 시험, 동아리) | 시간 배분 |
| 평일 저녁 | 모바일에서 오늘 한 일 체크, 내일 준비 확인 | 진도 확인 |
| 시험 2주 전 | 과목별 공부 일정 정리, 데드라인 우선순위화 | 집중 계획 |
| 과제 제출 직전 | 마지막 체크리스트 확인 | 누락 방지 |

### 기대하는 앱의 모습
- **정보 통합:** 모든 일(과제, 시험, 동아리, 개인)을 한 곳에서 관리
- **장기 추적:** 한 달 단위로 계획을 세우고 진행 상황을 보고 싶음
- **카테고리 체계:** 과목별, 활동별로 깔끔하게 분류
- **편한 검색:** 언제든 특정 카테고리의 모든 일을 한눈에 볼 수 있음

## 설계 원칙 (긴장점 해결)

### P1: 최소 클릭으로 추가 가능 (민준 ↔ 지은)
**민준의 요구:** 모바일에서 3초 이내에 추가  
**지은의 요구:** 카테고리, 우선순위까지 정확히 입력  
**해결:** 기본값(카테고리='일반', 우선순위='중')으로 빠르게 추가하되, 선택적으로 상세 입력 가능

### P2: 모바일과 데스크톱 동등한 경험 (민준과 지은 공통)
**달성 방법:** 반응형 디자인으로 모바일에서는 수직 스크롤, 데스크톱에서는 넓은 화면 활용  
**기능은 양쪽에서 동일하게 제공**

### P3: 카테고리로 정보를 체계화 (지은의 요구)
**달성 방법:** 사용자가 자유롭게 카테고리를 만들고 할당 가능  
**카테고리별 필터링으로 "지금 보고 싶은 일"만 노출**

### P4: 시각적 우선순위 파악 (둘 다)
**달성 방법:** 색상과 배치(위에 높은 우선순위)로 중요도 한눈에 인지  
**정렬 기능으로 우선순위 순서 유지**
